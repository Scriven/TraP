GUIDE TO LOADING DATABASE

- Uncompress the Transcribed Proteome (TraP) database tables: 
tar -- lzma -xvf TraP<date>.sql.tar.lzma
- Create the database and load the database tables: 
mysql -p -u root -e "CREATE DATABASE trap;" 
cat TraP<date>.sql | mysql -p -u root trap
